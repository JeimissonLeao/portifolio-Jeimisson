# 📊 CallCenter Analytics Pro — Dashboard de Atendimento ao Cliente

## 🏢 Sobre o Projeto
Projeto profissional de análise de dados desenvolvido em Microsoft Excel avançado, simulando
o ambiente real de uma Central de Atendimento ao Cliente (Call Center).
O projeto consolida 350 registros de atendimento do ano de 2024, aplicando técnicas de
Power Query, Tabelas Dinâmicas, fórmulas avançadas e dashboard executivo.

## 🎯 Objetivos
- Controle e análise de KPIs operacionais de atendimento
- Identificação de gargalos e oportunidades de melhoria
- Visualização de tendências mensais e comparativos por canal/agente
- Suporte à tomada de decisão com dados confiáveis e atualizados

## 📁 Estrutura do Arquivo Excel
| Aba | Descrição |
|-----|-----------|
| BASE_DADOS | 350 registros com dados de atendimento (protocolo, data, canal, TMA, SLA, NPS...) |
| KPIs | Painel com indicadores calculados: TMA, Taxa de Resolução, NPS, Desempenho por Agente |
| ANALISE_MENSAL | Consolidação mensal com gráficos de linha e barras |
| DASHBOARD | Painel executivo visual com cards de KPI e gráficos |
| DOCUMENTAÇÃO | Registro técnico de fórmulas, Power Query e glossário |

## 📊 KPIs Monitorados
- **TMA** — Tempo Médio de Atendimento (meta: < 8 min)
- **Taxa de Resolução** — % chamados resolvidos (meta: > 70%)
- **SLA** — % atendimentos dentro do prazo (meta: > 85%)
- **NPS** — Média de satisfação do cliente (escala 1–10)
- **Volume** — Total de atendimentos por período/canal/agente

## 🛠️ Tecnologias e Recursos Utilizados
- **Power Query** — ETL: limpeza, transformação e padronização dos dados
- **Tabelas Dinâmicas** — Análise multidimensional por canal, agente e período
- **Fórmulas Avançadas** — XLOOKUP, SOMASES, CONT.SES, MÉDIASES, SE aninhado
- **Formatação Condicional** — Semáforos visuais para SLA e satisfação
- **Gráficos Dinâmicos** — Linha, barras empilhadas, pizza com rótulos
- **Named Ranges e Tabelas Estruturadas** — Para fórmulas dinâmicas e escaláveis

## 📈 Principais Análises
1. Evolução mensal do volume de atendimentos
2. Comparativo de TMA e resolução por canal de atendimento
3. Ranking de desempenho por agente (TMA, resolução, NPS)
4. Distribuição de status (resolvido, pendente, cancelado, escalado)
5. Análise de SLA por canal e identificação de gargalos

## 📌 Como Usar
1. Abra o arquivo .xlsx no Microsoft Excel 2016 ou superior
2. Acesse a aba DASHBOARD para uma visão geral executiva
3. Use a aba BASE_DADOS para filtros e exploração detalhada
4. Consulte KPIs para indicadores por canal, status e agente
5. Atualize os dados na BASE_DADOS e pressione Ctrl+Alt+F5 para atualizar tudo
